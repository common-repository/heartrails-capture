== Description ==

Add website screenshot using [HeartRails Capture API](http://capture.heartrails.com/help/use_api). This is an unofficial HeartRails plugin. 

= Shortcode =

[capture url="http://example.com"]


== Installation ==

1. Upload capture-heartrails folder to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
